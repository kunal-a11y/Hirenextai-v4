Build Advanced Profile System and Improve Job Match Quality for HirenextAI Job Portal.

Project Context:
This is a full stack job portal built using Node.js (Express), React (Vite), and PostgreSQL.
Existing features include authentication, job search with filters, AI recommended jobs, support ticket system, admin panel, and AI credit system.

Goal:
Improve user profile quality so AI job recommendations become more accurate and user engagement increases.

==============================
STEP 1 — DATABASE UPGRADE
=========================

Update the profiles table and add new structured career fields.

Run SQL:

ALTER TABLE profiles
ADD COLUMN IF NOT EXISTS skills text[] DEFAULT '{}',
ADD COLUMN IF NOT EXISTS education jsonb DEFAULT '[]',
ADD COLUMN IF NOT EXISTS experience jsonb DEFAULT '[]',
ADD COLUMN IF NOT EXISTS career_goal text,
ADD COLUMN IF NOT EXISTS resume_url text,
ADD COLUMN IF NOT EXISTS profile_completion int DEFAULT 0;

Create indexes:

CREATE INDEX IF NOT EXISTS idx_profiles_completion ON profiles(profile_completion);
CREATE INDEX IF NOT EXISTS idx_profiles_skills ON profiles USING GIN (skills);

==============================
STEP 2 — PROFILE COMPLETION ENGINE
==================================

Create backend utility file:
utils/profileCompletion.ts

Logic:

Profile completion formula:

Basic info (name, email, location) = 20%
Skills added = 25%
Education timeline added = 15%
Experience timeline added = 20%
Resume uploaded = 10%
Career goal filled = 10%

Return percentage and update profile_completion column whenever profile is updated.

==============================
STEP 3 — RESUME UPLOAD + AI PARSING
===================================

Create API route:

POST /api/profile/upload-resume

Features:

* Accept PDF upload using multer
* Extract text using pdf-parse
* Send text to AI model
* AI must return structured JSON:

{
skills: [],
education: [],
experience: [],
careerGoal: ""
}

* Update user profile automatically
* Store resume_url
* Add validation
* Handle errors safely
